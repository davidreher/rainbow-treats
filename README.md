# Rainbow Treats

Mobile-first match-three puzzle Progressive Web App.

![CI](https://github.com/davidreher/rainbow-treats/actions/workflows/ci.yml/badge.svg)

Play on GitHub Pages: https://davidreher.github.io/rainbow-treats/

## Local Development

- `npm run dev` starts a local server at `http://localhost:4173`
- `npm run test:unit` runs the Vitest unit suite
- `npm run test:e2e` runs the Playwright end-to-end suite
- `npm test` runs unit and end-to-end tests
